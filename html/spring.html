We can calculate the energies of a spring \[x = \cos(\omega t+\phi)\] \[\dot{x} = -\omega\sin(\omega t+\phi)\] \[L = KE - PE = \frac{1}{2}m\dot{x}^2-\frac{1}{2}kx^2\] \[L = \frac{1}{2}m\omega^2\sin^2(\omega t+\phi)-\frac{1}{2}k\cos^2(\omega t+\phi)\] \[S = \frac{1}{2}m\omega^2\int_0^4\sin^2(\omega t+\phi) dt - \frac{1}{2}k\int_0^4\cos^2(\omega t+\phi) dt\] \[S = \frac{1}{2}m\omega^2\left(\frac{\omega t+\phi}{2\omega}-\frac{\sin 2(\omega t+\phi)}{4\omega}\right)\Bigg|_0^4 - \frac{1}{2}k\left(\frac{\omega t+\phi}{2\omega}+\frac{\sin 2(\omega t+\phi)}{4\omega}\right)\Bigg|_0^4\] \[S = \frac{1}{2}m\omega^2\left(\frac{4\omega}{2\omega}+\frac{\phi}{2\omega}-\frac{\sin 2(4\omega+\phi)}{4\omega}-\frac{\phi}{2\omega}+\frac{\sin 2\phi}{4\omega}\right) - \frac{1}{2}k\left(\frac{4\omega}{2\omega}+\frac{\phi}{2\omega}+\frac{\sin 2(4\omega+\phi)}{4\omega}-\frac{\phi}{2\omega}-\frac{\sin 2\phi}{4\omega}\right)\] \[S = \frac{1}{2}m\omega^2\left(2-\frac{\sin 2(4\omega+\phi)}{4\omega}+\frac{\sin 2\phi}{4\omega}\right) - \frac{1}{2}k\left(2+\frac{\sin 2(4\omega+\phi)}{4\omega}-\frac{\sin 2\phi}{4\omega}\right)\] \[S = \frac{m\omega}{8}\left(8\omega-\sin 2(4\omega +\phi)+\sin 2\phi\right) - \frac{k}{8\omega}\left(8\omega+\sin 2(4\omega +\phi) -\sin 2\phi\right)\]
